## What I Build

**AI & Agentic Systems**
- Streaming AI pipelines with SSE, tool-calling, and inline citations
- Perplexity-style search with semantic caching via OpenRouter
- Plan-then-execute agent flows (similar to Manus/Devin) with durable step checkpointing via Inngest
- pgvector RAG with team-level data silos and connector registry systems
- Custom Access Token Hooks and role-based JWT injection with Supabase

**Full-Stack Web Applications**
- Multi-tenant SaaS platforms with complex RBAC and middleware-gated routes
- Real-time features using Supabase Realtime and WebRTC signaling
- Custom email delivery pipelines (Nodemailer + Hostinger SMTP, Postgres function hooks)
- Enterprise admin panels with dashboards, HR suites, and admissions pipelines
- WebRTC video meeting apps with screen share renegotiation and TURN/STUN via Metered.ca

**UI/UX Engineering**
- Polished, animated interfaces with Framer Motion and precise geometric SVG components
- Glassmorphism, editorial serif aesthetics, and ghost-line timeline styling
- Multi-subdomain cookie scoping and production-hardened auth flows
- Data visualisation: animated funnels, gazette cards, chart exports via ExcelJS
